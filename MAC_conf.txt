Per configurare il Mandatory Access Control (MAC) per rendere il client-200 sensitive, viene utilizzando il modulo di sicurezza MAC Apparmor.

Descrizione di Apparmor:
	Implementa una politica MAC (Mandatory Access Control) incentrata sul task, in cui singoli programmi possono avere il proprio profilo.
		• Un profilo è un confinamento di un programma a un insieme di file, capacità, accesso alla rete e limiti di risorse.
		• I profili dei compiti vengono creati e caricati da userspace.
		• I compiti che non hanno un profilo eseguono in uno stato non confinato, equivalente al DAC (Discretionary Access Control) tradizionale di Linux (confinamento selettivo).



1) entrare nella directory etc/apparmor.d/
	-)In tale directory vengono definiti le policy profili di sicurezza per i programmi

2) le policy devono essere scritte secondo la sintassi Apparmor:
	2.1) creare un file scrivendo vi <FilenamePolicies>
	2.2)La sintassi da seguire è la seguente:
	#include <tunables/global>

	/bin/bash {
	  # Regole di accesso per il comando /bin/bash
	  network raw,
	  capability setuid,
	  /home/** rw,
	}

	# Altri profili di sicurezza possono essere definiti qui
